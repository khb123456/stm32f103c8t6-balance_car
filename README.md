# STM32F103C8T6 Self-Balancing Vehicle Project
> 基于STM32F103C8T6的两轮自平衡小车项目（开发中）

## 🚀 项目简介
本项目基于STM32F103C8T6单片机，实现两轮自平衡小车的直立控制、速度控制、方向控制，采用串级PID算法，搭配MPU6050姿态传感器、直流电机驱动模块。

## 🛠️ 硬件清单（开发中，持续更新）
- 主控：STM32F103C8T6 最小系统板
- 姿态传感器：MPU6050 六轴陀螺仪/加速度计
- 电机驱动：TB6612 直流电机驱动模块
- 电机：7.4V直流减速电机（带编码器）
- 电源：12V 锂电池 / 12V 电源适配器
- 其他：0.96寸OLED屏幕、JDY-31蓝牙模块、HC-SR04超声波模块、5V降压模块、轮趣R1平衡车套件

## 💻 软件环境
- 开发环境：Keil MDK 5.42.0.0
- 代码生成：STM32CubeMX
- 版本控制：Git + GitHub
